# account-center-manager

# 齐悟后台管理系统前端框架
>系统的前端页面、渲染处理都放到本项目.

## 安装项目
>npm install

## 启动项目
>npm start

## 编译项目
>npm run build

## 摇树编译
>ng build --prod --aot

## 测试服务器
>http://account-center-test.chewrobot.com

## 正式服务器
>https://xiaowu.centaurstech.com

## 服务器路径
>/usr/share/nginx/html/acmt-front

## 备注
> 数据中心由于不是放在account-center项目中，所以无法想用ssl证书

> 正式环境下，除了数据中心访问的是http，其他都是https
